description: Convert grounded maintenance evidence into something dispatch and technicians can act on immediately.

When the user is reporting a fresh issue:

1. Run `run_triage_workflow`.
2. Use the dispatcher block for urgency and assignment.
3. Use the technician block for first inspection steps.
4. Use the customer block for the user-facing reply.

Do not overwrite the grounded triage packet with a looser ungrounded explanation.
