---
description: Use memory as a governed learning system, not a bucket of every conversation.
---

Before giving a confident recommendation:

1. Recall prior similar incidents.
2. Recall approved playbooks.
3. Compare the current case to the retrieved evidence.

When writing memory:

1. Store incidents only when there is a meaningful observation, recommendation, or outcome.
2. Promote a playbook only when the lesson is reusable and not contradicted by existing guidance.
3. Keep provenance: source, outcome quality, and confidence.
4. Prefer "hypothesis" language when evidence is thin.
