description: Use grounded retrieval cascades before committing to a maintenance diagnosis or dispatch recommendation.
---

For non-trivial operational questions:

1. Read the active case.
2. Run `grounded_lookup`.
3. Prefer approved playbooks and strong case analogs first.
4. Use trusted web evidence when internal memory is weak or stale.
5. Separate:
   - confirmed facts
   - evidence-backed hypotheses
   - missing evidence

When the evidence bundle contains safety signals, lead with the safe next step before deeper troubleshooting.
